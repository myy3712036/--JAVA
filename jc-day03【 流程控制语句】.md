# 2.选择结构之if语句//记忆

#### 概述:选择结构,用来解决现实生活中的选择问题,

#### 例如如果满足一个条件就做什么,否则又做什么.

~~~java
格式1:一选一的情况,或者多选多的情况
if(条件){//条件结果是布尔类型,条件满足结果是true,条件不满足结果就是false,
	//只有上面条件满足,结果为true的时才进来这里面做事情(你写的代码),如果条件不满足结果是false的时不进来
}

格式2:二选一的情况,两者只能选其一,不是你死就是我活的情况
if(条件){//条件结果是布尔类型,条件满足结果是true,条件不满足结果就是false,
	//只有上面条件满足,结果为true的时候才进来这里面做事情(你写代码),下面的else不走,
	//如果条件不满足结果是false的时候不进来
}else{
	//如果上面的条件不满足结果是false的时候,就进来这里做事情
}

格式3:多选一的情况,多种条件,我只能做一个事情
if(条件){//条件结果是布尔类型,条件满足结果是true,条件不满足结果就是false,
	//只有上面条件满足,结果为true的时候才进来这里面做事情(你写代码),下面不走,
	//如果条件不满足结果是false的时候不进来
}else if(条件){//elseif不能省略,但是可以有多个
	//只有上面条件满足,结果为true的时候才进来这里面做事情(你写代码),下面不走,
	//如果条件不满足结果是false的时候不进来
}else{//可以省略的,但是建议大家不要省略,因为可以给出额外的提示
	//只有上面的条件都不满足,才能进来这里面做事情
}
~~~


# 3.选择结构之switchcase语句//记忆

#### 概述:switch用来筛选的,case就是情况(条件),用来筛选某个情况,

#### 如果符合某个情况就做某个事情,有点类似if语句

~~~java
//格式标准代码使用如下:
public static void main(String[] args){
	//模拟幸运大抽奖
	int key = 22;
	//筛选变量key的值,如果满足case情况后面的值跟它相等,就进来case里面做事情,不相等就不进来,
    //层层筛选,从上往下
	switch (key) {
    //key可以是常量,也可以是变量名,
    //如果是变量名,它的数据类型有要求:byte,short,char,int,jdk1.5支持枚举,jdk1.7开始支持字符串String类
    case 1://case后面跟的只能是常量,并且如果有多个case后面常量的值不能相同,不能用变量
        System.out.println("恭喜你中了一台电视机");
        break;//跳出的意思,用来结束switch语句,下面的代码不走

    case 2://case后面跟的只能是常量,并且如果有多个case后面常量的值不能相同,不能用变量
        System.out.println("恭喜你中了一台电冰箱");
        break;//跳出的意思,用来结束switch语句,下面的代码不走

    default://默认情况下,如果上面的情况都不满足,那就进来这里面做事情,
            //有点类似if...elseif...else里面的else,default可以省略,但是不建议,因为可以给出额外的提示
        System.out.println("很遗憾你木有中奖,给你一个飞吻,么么哒");
        break;
    }  
}
~~~

~~~java
面试题:
1.default可以省略吗?可以,但是不建议,因为可以给出额外的提示;
  default一定要放在最后面吗?不一定,可以放在switch语句的合法位置
  
2.case后面能跟变量吗?不能,只能是常量,如果有多个case,要求常量不能相同

3.break可以省略吗?最后一个break可以省略(switch语句的结束条件要么遇到break要么遇到switch语句的}),
  其他的最好不要省略,否则会出现case穿透造成你不想要的数据

4.case穿透:没有遇到break,也没有遇到switch语句的},下面的case或者default不用判断直接执行里面的东西,
  直到(switch语句的结束条件要么遇到break要么遇到switch语句的})才结束,停止
  如果多个case里面要做的事情是一样的,那就可以利用case穿透提高代码的复用性(少写了很多代码实现一样的功能)

5.什么时候使用switch语句?什么时候使用if语句?
  因为switch语句的case的值只能是常量,固定值,如果是固定值,推荐使用switch语句,
  如果不是固定的值,推荐使用if语句,如果你不知道使用什么语句,就用if语句!!!
~~~



# 4.循环结构

#### 概述:循环就是不断的做事情,可以有一定的终止条件,也可以不终止(叫死循环,根本停不下来)

~~~java
分类:用三种不同的循环,在控制台输出1到10
	int i = 1;//初始化条件
	while(i<=10){//判断条件
		System.out.println(i);//循环体,就是你要做的事情
		i++;//控制条件
	}
	执行流程:首先执行初始化条件一次,然后执行判断条件,条件满足结果是true的时候进来执行循环体,
	我想改变条件,执行控制条件,条件就改变了,就要重新判断条件,条件满足就再次进来,
	如果条件(一开始)都不满足结果是false就不进来执行循环,循环结束了
	
	int i = 1;//初始化条件
	do {
		System.out.println(i);//循环体,就是你要做的事情
		i++;//控制条件
	}
	while (i<=10);//判断条件
	
	执行流程:首先执行初始化条件一次,直接执行循环体,
	想要条件改变,执行控制条件,条件改变,就要判断条件是否满足,
	如果条件满足结果是true,就继续执行循环体,执行控制条件,
	如果条件不满足结果是false那就不再进来执行循环,循环结束了(do...while至少多执行了一次循环体,控制条件)
	
	for(int i = 1;i<=10;i++){//int i = 1;初始化条件	i<=10;判断条件	i++控制条件
		System.out.println(i);//循环体,就是你要做的事情
	}	
	执行流程:首先执行初始化条件一次,然后执行判断条件,条件满足结果是true的时候进来执行循环体,
	我想改变条件,执行控制条件,条件就改变了,就要重新判断条件,条件满足就再次进来,
	如果条件(一开始)都不满足结果是false就不进来执行循环,循环结束了
	
总结:循环就是不断的做事情,当条件不满足的时候停止结束循环,如果条件一直满足就是死循环,根本停不下来!!!
~~~

```java
面试题:
	1.do...while循环跟其他两种循环的区别:
	  do...while一开始不用判断条件直接执行循环体,
	  其他两种循环都要先判断条件成立再进来执行循环体,不成立不进来循环就结束了,
	  do...while比其他两种循环至少多执行了一次循环体

	2.for循环与其他两种循环的区别:
	  for循环里面定义的变量在循环的外面不能继续访问,循环使用完毕就从内存中消失,提高了内存的使用效率,
	  其他两种循环的变量定义在循环的外面,循环结束后还可以继续使用

	3.什么时候使用for循环,什么时候使用while循环?
	  如果循环次数固定的时候推荐for循环,因为for循环里面定义变量在循环的外面不能继续访问,
	  循环使用完毕就从内存中消失,提高了内存的使用效率,
	  如果循环的次数不固定推荐使用while循环,如果你不知道使用哪种循环,就用for循环,原因同上

	4.你能写出三种不同的死循环吗?(循环就是不断的做事情,死循环比较特殊,不终止,一直做)
	  while (true) {
		//循环体,就是你要做事情
	  }

	  do {
		//循环体,就是你要做事情
	  } while (true);

	  for (;; ) {
		//循环体,就是你要做事情
	  }
```
~~~java
循环控制跳转语句:
break://表示跳出整个循环,循环不再执行,下面的代码不走
代码:
public static void main(String[] args){
	for (int i = 1; i <=5; i++) {//5.for
		if (i==4) {
			break;//表示跳出整个循环,循环不再执行,下面的代码不走
		}

		System.out.println(i);//1,2,3
	}
}

continue://跳过本次循环下面的代码不走,继续下一次循环
代码:
public static void main(String[] args){
	for (int i = 1; i <=5; i++) {
		if (i==4) {
			continue;//表示跳出本次循环,循环继续执行,本次不执行下面代码不走,其他次执行
		}
			
		System.out.println(i);//1,2,3,5
	}
}
~~~
